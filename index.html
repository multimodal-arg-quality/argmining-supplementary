Key Definitions

TermDefinition
MAGMultimodal Argument Graph — graph-based representation with claim, evidence, and backing nodes connected by warrant, complementarity, and support edges.
L1–L4Four-level semantic content model (after Lundgard & Satyanarayan, 2021): L1 = chart structure, L2 = statistical patterns, L3 = perceptual salience, L4 = semantic patterns.
Toulmin ModelExtended framework: Claim (text), Evidence (visual), Warrant (cross-modal reasoning), Backing (authoritative source), Qualifier (scope/certainty), Rebuttal.
Q(G)Composite quality score: Q(G) = Compgate × [0.20×Align + 0.30×WStr + 0.30×SRel + 0.20×CBudget] / 5
TiersTop (Q ≥ 3.5), Mid (2.5 ≤ Q < 3.5), Bottom (Q < 2.5)
